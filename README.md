# Template para aplicações Angular

# Instalação
* Possuir Angular e Angular CLI instalado
* Possuir NodeJS instalado
* Clonar este reposítorio
* Executar os comandos:

```cmd
npm install
npm start
```
* Acessar: http://localhost:4200
